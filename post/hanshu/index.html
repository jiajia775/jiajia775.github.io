Go语言基础之函数

2019-09-03

函数是组织好的、可重复使用的、用于执行指定任务的代码块。

函数的定义与调用\color{red}{函数的定义与调用}函数的定义与调用

Go语言的函数定义使用func关键字定义
具体格式如下：

func 函数名(函数参数)函数返回值{
	函数体
}

定义一个简单的两数相加的函数：
package main

import "fmt"

func calc(a, b int) int {
	return a + b
}

func main() {
	result := cala(1, 2)
	fmt.Println(result) //3
}

Go语言推荐写法如下：
即给函数返回值命名

package main

import "fmt"

func calc(a, b int) (result int) {
	result = a + b
	return
}

func main() {
	result := cala(1, 2)
	fmt.Println(result) //3
}

函数的参数和返回值都是可选的，可以实现一个既不需要参数也没有返回值的函数：

package main

import "fmt"

func calc() {
	fmt.Println("jiajia")
}

func main() {
	cala()
}

参数\color{red}{参数}参数

可变参数

可变参数是指函数的参数数量不固定。Go语言中的可变参数通过在参数名后加...来标识。
注意：可变参数通常要作为函数的最后一个参数

func calc(x ...int) (result int) {
	result = 0
	for _, v := range x {
		result = result + v
	}
	return
}

调用：

func main() {
	result1:= calc()
	result2 := calc(10, 20)
	result3 := calc(10, 20, 30)
	fmt.Println(result1) //0
	fmt.Println(result2) //30
	fmt.Println(result3) //60
}

固定参数搭配可变参数使用时，可变参数要放在固定参数的后面，示例代码如下：

func calc(x int, y ...int) (result int) {
	result = x
	for _, v := range y {
		result = result + v
	}
	return
}

调用：

func main() {
	result1 := calc(10)
	result2 := calc(10, 20)
	result3 := calc(10, 20, 30)
	fmt.Println(result1) //10
	fmt.Println(result2) //30
	fmt.Println(result3) //60
}

变量作用域\color{red}{变量作用域}变量作用域

全局变量

全局变量是定义在函数外部的变量，它在程序整个运行周期内都有效。 在函数中可以访问到全局变量。

package main

import "fmt"

var a string = "jiajia"

func calc() {
	fmt.Println(a) //"jiajia"
}

func main() {
	calc()
}

局部变量

  • 函数内定义的变量无法在该函数外使用
package main

import "fmt"

func calc() {
	//定义一个函数局部变量a,仅在该函数内生效
	var a int = 10
	fmt.Println(a)
}

func main() {
	calc()
	fmt.Println(a) //error,此时无法使用变量x
}
  • 语句块定义的变量
package main

import "fmt"

func calc() {
	for i := 0; i < 10; i++ {
		fmt.Println(i)
	}
	fmt.Println(i)//此处无法使用变量i
}

func main() {
	calc()
}

注意：如果局部变量和全局变量重名，优先访问局部变量。

函数类型与变量\color{red}{函数类型与变量}函数类型与变量

定义函数类型

type Functype func(int, int) int
其中，Functype是函数类型，它是一种函数类型，这种函数接收两个int类型的参数并且返回一个int类型的返回值。

package main

import "fmt"

func Add(a, b int) int {
	return a + b
}

func Min(a, b int) int {
	return a - b
}

//函数也是一种数据类型，通过type可以给函数类型起名字
//FuncType就是这个函数类型所起的名字

type funcType func(int, int) int

func main() {
	add := Add(10, 20)
	fmt.Println("add = ", add) //传统方式

	//声明函数类型的变量，变量名叫做fTest
	var fTest funcType
	fTest = Add                 //是变量名就可以赋值 也可以统一写成 var fTest FuncType = Add
	result01 := fTest(100, 200) //用result01来接收
	fmt.Println("result01 = ", result01)

	fTest = Min
	result02 := fTest(200, 50)
	fmt.Println("result02 = ", result02)
}

高阶函数\color{red}{高阶函数}高阶函数

函数作为参数(回调函数)

package main

import "fmt"

func Add(a, b int) int {
	return a + b
}

// Functype
type Functype func(int, int) int

//函数中有一个参数是函数类型，这个函数就是回调函数
//多态，多种形态，也就是说调用同一个接口，可以实现不同的表现，
func calc(a, b int, op Functype) int { //op相当于变量名可以写成：op func(int,int)int
	return op(a, b)
}

func main() {
	result := calc(10, 20, Add)
	fmt.Println(result)
}

匿名函数和闭包\color{red}{匿名函数和闭包}匿名函数和闭包

匿名函数

匿名函数就是没有函数名的函数，匿名函数的定义格式如下：

func(参数)(返回值){
    函数体
}

匿名函数多用于实现回调函数和闭包。
示例：

package main

import "fmt"

func main() {
	a := 10
	str := "jiajia"
	//匿名函数，函数没有名字，
	//1.0
	f1 := func() {
		fmt.Println("a = ", a)
		fmt.Println("str = ", str)
	}
	f1() //函数调用

	//2.0
	func() {
		fmt.Println("a = ", a)
		fmt.Println("str = ", str)
	}() //可以并不需要参数接收，自己直接调用

	//3.0
	type FuncType func()
	var f3 FuncType //声明变量赋值
	f3 = f1
	f3()

	//4.0有参无返回值的匿名函数
	func(a, b int) {
		fmt.Printf("a = %d,str = %d\n", a, b)
	}(100, 200)

	//5.0有参有返回值的匿名函数
	result := func(a, b int) (result int) {
		result = a + b
		return
	}(3, 4)
	fmt.Println("result = ", result)
	//等同于下面的写法
	var result2 int
	result2 = func(a, b int) (result2 int) {
		result2 = a + b
		return
	}(3, 4)
	fmt.Println("result2 = ", result2)
}

闭包

闭包指的是一个函数和与其相关的引用环境组合而成的实体。简单来说，闭包=函数+引用环境，它不关心这些捕获了的变量和常量是否超出作用域，只要闭包还在使用它，这些变量都是会存在的
简单例子：

package main

import "fmt"

func main() {
	a := 10
	str := "java"

	func() {
		a = 6
		str = "go"
		fmt.Printf("内部:a = %d,str = %s\n", a, str) //内部:a = 6,str = go
	}()
	fmt.Printf("外部:a = %d,str = %s\n", a, str) //外部:a = 6,str = go
}

复杂例子：

package main

import "fmt"

//函数的返回值是一个匿名函数，返回一个函数类型
func MyFunc02() func() int {
	var x int

	return func() int {
		x++
		return x * x
	}

}

func MyFunc01() int {
	//函数被调用的时候才开始分配空间。才初始化为0

	var x int //没有初始化，值为0
	x++
	return x * x //函数调用完毕，x自动释放
}

func main() {
	//1.0
	fmt.Println(MyFunc01()) //1
	fmt.Println(MyFunc01()) //1
	fmt.Println(MyFunc01()) //1
	fmt.Println(MyFunc01()) //1

	//2.0
	//因为返回的是一个函数类型，需要一个变量来接收。通过f来调用返回的匿名函数，f来调用闭包
	//他不关心这些捕获了的变量或者常量是否超出作用域
	//只要闭包还在使用它，这些变量就会存在
	f := MyFunc02()
	fmt.Println(f())
	fmt.Println(f())
	fmt.Println(f())
	fmt.Println(f())
	fmt.Println(f())
}

defer语句\color{red}{defer语句}defer语句

Go语言中的defer语句会将其后面跟随的语句进行延迟处理。

func main() {
	fmt.Println("start")
	defer fmt.Println(1)
	defer fmt.Println(2)
	defer fmt.Println(3)
	fmt.Println("end")
}

输出结果：

start
end
3
2
1

由于defer语句延迟调用的特性，所以defer语句能非常方便的处理资源释放问题。比如：资源清理、文件关闭、解锁及记录时间等。
defer案例题：

package main

import "fmt"

func calc(index string, a, b int) int {
	ret := a + b
	fmt.Println("-----------------")
	fmt.Println(index, a, b, ret)
	return ret
}

func main() {
	a := 1
	b := 2				//3---->4
	defer calc("1", a, calc("10", a, b))
	a = 0				//2---->2
	defer calc("2", a, calc("20", a, b))
	b = 1
}

输出结果：

----------
10 1 2 3
-----------------
20 0 2 2
-----------------
2 0 2 2
-----------------
1 1 3 4
