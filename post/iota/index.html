Go语言中iota的基本使用

iota\color{red}{iota}iota

iota是go语言的常量计数器，只能在常量的表达式中使用
iota在const关键字出现时将被重置为0。const中每新增一行常量声明将使iota计数一次

const (
	n1 = iota //0
	n2        //1
	n3        //2
	n4        //3
)

几个常见的iota示例:\color{red}{几个常见的iota示例:}几个常见的iota示例:

  • 使用_跳过某些值
const (
	n1 = iota //0
	n2        //1
	_
	n4 //3
)
  • iota声明过程中间插队
const (
	n1 = iota //0
	n2 = 100  //100
	n3 = iota //2
	n4        //3
)
const n5 = iota //0
  • 多个iota定义在一行
const (
	a, b = iota + 1, iota + 2 //1,2
	c, d                      //2,3
	e, f                      //3,4
)
